# Protif-lio_Dev
# 🚀 Portfólio Dev Premium

Um portfólio moderno, responsivo e interativo desenvolvido para apresentar projetos, habilidades e serviços de desenvolvimento de software de forma profissional e visualmente impactante.

O projeto foi construído utilizando tecnologias web modernas e efeitos visuais avançados para proporcionar uma experiência elegante, fluida e responsiva em qualquer dispositivo.

---

# ✨ Funcionalidades

- 🎨 Interface moderna estilo premium/cyberpunk
- 📱 Responsividade Mobile First
- 🌌 Fundo animado com partículas
- ⚡ Efeitos visuais com Glassmorphism
- 🔥 Animações suaves com CSS
- 🧠 Texto animado com Typed.js
- 🍔 Menu hambúrguer responsivo
- ⬆️ Botão "Voltar ao topo"
- 📂 Seção de projetos/portfólio
- 🛠️ Cards interativos de serviços
- 📬 Formulário funcional com EmailJS
- 🌙 Dark Mode moderno
- 🎯 Navegação suave entre seções

---

## Front-end

- JavaScript
